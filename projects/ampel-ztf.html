post_candidate(view, filters=None, groups=None, instrument=None)¶
Perform the following actions:

  • Post candidate to filters specified by filters. ra/dec are taken from the most recent detection; drb from the maximum over detections.

  • Post photometry using the candid of the latest detection.

  • Post a PNG-encoded cutout of the last detection image.

  • Post each T2 result as comment with a JSON-encoded attachment. If a comment corresponding to the T2 unit already exists, overwrite it with the most recent result.

Parameters

  • view (TransientView) – Data to post

  • filters (Optional[List[str]]) – Names of the filter to associate with the candidate. If None, use filters named AMPEL.{channel} for each channel the transient belongs to.

  • groups (Optional[List[str]]) – Names of the filter to associate with the candidate. If None, use all accessible groups associated with token.

  • instrumentname – Name of the instrument with which to associate the photometry.
